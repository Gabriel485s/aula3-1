Utilizando blockquote.

Finalidade da tag BLOCKQUOTE

>
BLOCKQUOTE cria um bloco separado (assim como o p = parágrafo faz), e recua um pouco o texto para que ele se pareça um pouco mais como uma citação. Para algum texto em se queira destacar, podemos utilizar este recurso. Já estamos sabendo muita coisa sobre HTML, não é?
